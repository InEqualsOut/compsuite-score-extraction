# CompetitionSuite Score Extraction
This repository contains code to extract WGI scores from CompetitionSuite for the purpose of visualization and analysis. This also serves as an attempt to preserve historical score records in tabular format.

# Usage
Currently, the functionality of this code is limited in scope. The `scores_and_ranks.ipynb` file is able to process certain shows as described below:
- marching percussion shows
- shows with a double panel
- shows that utilize the Effect-Music / Effect-Visual split instead of the General Effect category (in effect since 2016)
- recaps that display the ensemble location in the second column
